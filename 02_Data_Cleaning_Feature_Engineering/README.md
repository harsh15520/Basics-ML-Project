# O2_Data-Cleaning-Feature-Engineering

## 📌 Overview
This repository contains a Jupyter Notebook focused on **data cleaning and feature engineering** for machine learning projects. It demonstrates practical preprocessing techniques that transform raw datasets into structured, model-ready inputs.

## 🎯 Objectives
- Handle missing values and outliers systematically.
- Apply normalization, scaling, and encoding strategies.
- Engineer new features to improve predictive performance.
- Document reproducible workflows for future ML pipelines.

## 🛠️ Tech Stack
- **Language:** Python 3 (ipykernel)
- **Environment:** JupyterLab
- **Libraries:** pandas, numpy, scikit-learn, matplotlib, seaborn
