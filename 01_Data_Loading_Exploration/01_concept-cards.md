These explain the foundational concepts behind Module 1:

01_data_loading

What: Loading data from disk to memory
Why: Garbage in → garbage out (if you load wrong, everything breaks)
When: First thing you do with any new dataset
Read this if: You want to understand why loading is critical


01_exploration_basics

What: df.shape, df.info(), df.describe() tell you about your data
Why: Each reveals different aspects (dimensions, types, ranges)
When: After loading, before any transformations
Read this if: You want to know what each diagnostic tool does


01_diagnosis_framework

What: The 3-part systematic diagnosis (missing → types → ranges)
Why: Order matters (you diagnose before fixing)
When: Applied consistently to ANY dataset
Read this if: You want a framework that works for every dataset
